# ROMA AETERNA EST - roma.mt

Website created with the help of www.11ty.dev, about the history of the Roman Empire, and Western Civilization.

# Generate pages
Information adapted from the [Getting Started Guide](https://www.11ty.dev/docs/)

1. Run `npm install @11ty/eleventy`.
2. Run `npx @11ty/eleventy` to generate the pages OR `npx @11ty/eleventy --serve` for local testing.
3. Generated pages contained inside the `_site` folder.
